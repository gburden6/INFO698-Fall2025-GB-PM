# INFO698-Fall2025-GB-PM
Repository for INFO Capstone Project
This repository contains the code and notebooks for the INFO 698 Capstone Project: an analysis of Student Term GPA data.
This repository and its contents were created by Grant Burden and Phillip McManus.

Different Machine Learning Models are split between different notebook files. 
INFO 698 Notebook has the KNN models, and the RF_Model Notebook contians the Rnadom Forest Regressor model.
the INFO698_DataViz notebook contains the code used for exploratory data analysis, statistical analysis of Term GPAs, and Grant's final visualizations used in the paper.
The gitignore file contians the the data set, which cannot be uploaded or shared. 

INFO698_McManus contains all code created by Phillip McManus, including data cleaning, linear regressions, model visualizations and all exploratory data analysis
with accompanying data visualizations to research the dynamics of first generation students and their GPA.
